Technology Stack

Explore the technologies powering our AI Chat Application.

Frontend Technologies

HTML5

Used for structuring the web page, including the sidebar, chat interface, and file upload section.

CSS3

Responsible for styling the interface, including the dark theme, gradients, and responsive layout.

JavaScript

Handles dynamic interactions, such as file uploads, toggling, and sending/receiving messages.

Marked.js

Parses Markdown content to render AI responses with proper formatting (headings, lists, code blocks).

DOMPurify

Sanitizes HTML content to prevent XSS attacks, ensuring safe rendering of AI responses.

Font Awesome

Provides icons for the UI, such as the ellipsis for dropdown menus.

Backend Technologies

Flask

A lightweight Python web framework used for routing, request handling, and response generation.

SQLite3

A lightweight database for storing user information (e.g., email, password).

Pillow (PIL)

Used for image processing, such as handling uploaded images.

pdf2image

Converts PDF files into images for text extraction, relying on Poppler for rendering.

Google Generative AI

Interacts with the Gemini 1.5 API to generate AI responses to user questions.

ZeroBounce API

Validates email addresses during user signup to ensure they are real and deliverable.

Additional Tools

Poppler

A PDF rendering library used by pdf2image to convert PDFs into images.

Microsoft Visual Studio Code

The primary code editor used for developing the application.

Concurrent Futures

Enables parallel processing of tasks, such as extracting text from multiple PDF pages.

Requests

Makes HTTP requests to external APIs, such as ZeroBounce for email validation.
